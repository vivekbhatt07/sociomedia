# SOCIOMEDIA

## Live Link - https://sociomedia.vercel.app/

A social media website where you can connect with your friends!

---

## INSTALLATION

```
1. git clone https://github.com/vivekbhatt07/sociomedia.git
2. cd sociomedia
3. npm install
4. npm run dev
```
